03 · sobre mim

Quem é a Mika

Oi! Sou Mikaelle, mas pode me chamar de Mika. Sou estudante de Ciência da Computação e desenvolvedora apaixonada pela intersecção entre tecnologia e criatividade. Fã declarada da Hello Kitty 🎀 e sobrevivente de código movida a cappuccino.

Minha maior força está no Java — back-end robusto, APIs REST e sistemas enterprise. É a linguagem que o mercado procura e eu entrego. No front, trabalho com React e Angular, além de HTML, CSS e JavaScript puro.

No mobile, uso Flutter e Dart pra criar experiências nativas incríveis. E tenho background em Design e Marketing, o que me faz pensar sempre em quem vai usar o que eu crio.

Um dos projetos que mais me orgulha é o SCAF — sistema full-stack desenvolvido em parceria com a Petrobras (Programa SAGA), com React no front e Java no back. Prova de que dá pra ir longe quando você mistura código com propósito. 🚀
